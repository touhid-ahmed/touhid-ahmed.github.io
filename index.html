Research Works

In essence, My research interest lies in developing various robotic system applications such as mapping, path planning, image processing, and end-effector manipulation on the Robot Operating System (ROS) platform.

A ROS-based Voice Controlled Robotic Arm for Automatic Segregation of Medical Waste Using YOLOv3

Authors : Samiha Tasnim, Zahid Hasan, Touhid Ahmed, Md. Tanvir Hasan, Ferdousi Jamal Uddin, Tanjila Farah

2nd International Conference on Computer, Control and Robotics, 2022

The work proposes a ROS based robotic arm for medical waste segregation using YOLOv3. The paper has been accepted in the International Conference on Computer, Control and Robotics (ICCCR-2022).

Autonomous Intruder Detection using a ROS-based Multi-Robot System equipped with 2D-LiDAR Sensors

Authors : Mashnoon Islam, Touhid Ahmed, Abu Tammam Bin Nuruddin, Mashuda Islam, Shahnewaz Siddique

IEEE International Symposium on Safety, Security, and Rescue Robotics (SSRR), 2020
Paper / Presentation

The work proposes a ROS based multi-robot intruder detection system using 2D-LIDAR data percieved from a team of robots. The robustness and efficiency of the central bot (MIDNet) has been tested and benchmarked using two test cases. The simulation results are shown here.

Dual-Order Resource Allocation in 5G H-CRAN Using Matching Theory and Ant Colony Optimization Algorithm

Authors : Shadman Sakib Arnob, Iftekharul Islam Shovon, Touhid Ahmed, Md. Sami Ullah, Rashed Shelim

46th Annual Conference of the IEEE Industrial Electronics Society (IECON), 2020
Paper / Presentation

The paper addresses the resource allocation problem in 5G small cell networks by proposing an optimization algorithm which combines the Matching Theory Algorithm and the Ant Colony Optimization Algorithm.

A Real-Time Controlled Closed Loop IoT Based Home Surveillance System for Android using Firebase

Authors : Touhid Ahmed, Abu Tammam Bin Nuruddin, Asif Bin Latif, Shadman Sakib Arnob, Rummana Rahman

6th International Conference on Control, Automation and Robotics (ICCAR), 2020
Paper / Presentation

A real-time home surveillance system is designed that can be authorised and regulated remotely with the help of an Android application.
Projects

e-Yantra Robotics Competition 2020-2021

Participant, 2020

Lead a team where we developed ROS packages for an Autonomous Ground Vehicle (AGV) to make it capable of autonomously traversing an indoor environment to assist moving objects from one place to another.

Lane detection using Python and OpenCV

Worked on a lane detection algorithm for detecting lane lines using python and OpenCV.

Red-colored object detection using ROS and OpenCV

This is an object detection project for detecting red colored objects through color segmentation using openCV and ROS. The CvBridge Library has been used for Image message conversion between OpenCV and ROS.

Arduino Based Bluetooth Controlled Gripper Bot

Worked on an Arduino based Bluetooth controlled gripper bot that can be remotely controlled with an android app. The bot can be used to pick up and drop objects of limited diameter. Project Demonstration

Arduino Based Obstacle Avoiding bot

Made an Arduino based obstacle avoiding robot using ultrasonic sensor. The robot is able to automatically sense the obstacle in front of it using ultrasonic sensor and avoid them by turning itself in another direction. Project Demonstration
